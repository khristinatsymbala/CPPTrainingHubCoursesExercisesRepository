C-style strings

Sequence of characters / Послідовність символів :
    Continuous in memory / Безперервна в пам'яті
        Пояснення:
        С-рядки представляють собою послідовність символів, що зберігаються безперервно в пам'яті. 
        Це означає, що всі символи рядка розміщуються один за одним без будь-яких проміжків між ними.

    Implemented as an array of characters / Реалізовані як масив символів
        Пояснення:
        С-рядки реалізовані у вигляді масиву символів. Кожен символ у рядку займає окрему комірку в масиві.
        Це дозволяє доступ до кожного символу за допомогою індексу масиву.

    Terminated by a null character (null) / Завершуються нульовим символом (null)
        Пояснення:
        Кожен С-рядок обов'язково закінчується нульовим символом (null character), який має значення нуля. 
        Цей символ вказує на кінець рядка. Без цього символу неможливо визначити довжину рядка, оскільки він не містить інформації про свою довжину окремо.

    Null character with value of zero / Нульовий символ зі значенням нуль
        Пояснення:
        Нульовий символ ('\0') є спеціальним символом, який має числове значення 0. 
        Цей символ використовується для позначення кінця С-рядка, щоб функції, які працюють з рядками, знали, де закінчується рядок.

    Referred to as zero or null terminated strings / Відомі як нуль-терміновані рядки
        Пояснення:
        С-рядки часто називають нуль-термінованими рядками, оскільки вони закінчуються нульовим символом. 
        Ця характеристика відрізняє їх від інших типів рядків, які можуть мати інші методи визначення довжини.

String literal / Рядковий літерал:

    Sequence of characters in double quotes, e.g., "Frank" / Послідовність символів у подвійних лапках, наприклад, "Frank"
        Пояснення:
        Рядковий літерал у С мовах - це послідовність символів, укладена в подвійні лапки. 
        Наприклад, "Frank" є рядковим літералом. Він створюється безпосередньо у вихідному коді.

    Constant / Константа
        Пояснення:
        Рядковий літерал є константою, тобто його значення не може бути змінене під час виконання програми. 
        Він зберігається в сегменті пам'яті, призначеному для зберігання констант.

    Terminated with null character / Завершується нульовим символом
        Пояснення:
        Як і всі С-рядки, рядкові літерали також завершуються нульовим символом. Наприклад, рядок "Frank" насправді зберігається як 'F', 'r', 'a', 'n', 'k', '\0' у пам'яті.
        Це забезпечує правильне функціонування стандартних бібліотечних функцій, які працюють з рядками.

declaring variables:
    char my_name[8];
    my_name = "Frank"; // Error
    strcpy(my_name,"Frank"); //OK
